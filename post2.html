Clinic Schedule Application

This was a project I started at April 2021 for a surgery clinic to mimic an online schedule website with more options.

The system provide the records of two months (current and next one) with the automatic saving of the records of the previous month once it overs.

The user has the options to go to any day just from calender and assign employee of that day.

User can set the number of available shefts for each day so it can track how many left after each assigned shift.

User can also reset all shift of specific day or postpone it to another day.
